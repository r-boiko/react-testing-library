# React Testing Library

### Описание:

Тестируем React-приложения и React-компоненты с помощью библиотеки react-testing-library. Связка React и React Router, а так же React, React-хуков и Redux.

## Используемые ресурсы и инструменты:

- [Visual Studio Code (редактор кода)](https://code.visualstudio.com)
- [Node.js](https://nodejs.org/en/)
- [Yarn](https://yarnpkg.com)
- [Jest (расширение для VS Code)](https://marketplace.visualstudio.com/items?itemName=Orta.vscode-jest)
